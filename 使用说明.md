# ZZZ DDS 贴图编辑器 · 使用说明

双击项目根目录的 **`ZZZ-DDS-Editor.exe`** 即可打开（绿色单文件，免安装；首次启动会自解压，稍等几秒）。

---

## 一、第一次使用：先设工作目录
1. 点右上角 **⚙ 设置**。
2. 在「工作目录」里点 **选择目录**，挑一个文件夹（编辑记录会自动存到这里，建议专门建一个，比如 `D:\zzz贴图记录`）。
3. 顺便可以在设置里调「默认导出格式」「自动保存」「素材文件夹」，关掉后再回主界面。

> 没设工作目录时，⚙ 设置按钮会蓝色描边提醒；打开 Mod 时也会先让你设。

## 二、打开要编辑的贴图
1. 点 **打开 Mod 文件夹**，选中那个 mod 的文件夹。
2. 弹出贴图选择框，里面列出该 mod 里所有 DDS（已经编辑过的会标「已有记录」角标）。点一张开始编辑。
3. 之后想继续上次的，直接 **最近记录**，或再次「打开 Mod 文件夹」选同一张 → 自动接着上次的图层续编。

## 三、准备要贴上去的素材
- 点右侧「待插入贴图」面板的 **选择文件夹**，选一个放素材（PNG / DDS）的文件夹。
- 缩略图按使用频率排序，用得越多越靠前。设置里记住后，下次启动自动加载。

## 四、把素材放到贴图上（两种方式）

**方式 A · 拖拽**：把右侧素材直接拖到画布，松手处就是该图层中心。

**方式 B · 两次点击放置**（适合精确定位 / 定角度）：
1. 在右侧 **点一下** 某个素材（选中后它会高亮）。
2. 在背景贴图上 **点第一下** = 这张新贴图的 **中心点**。
3. **移动鼠标**：当前鼠标位置 = 新贴图的 **右下角**，移动时实时预览大小和角度。
4. **再点一下** = 固定。按 `Esc` 可随时取消。

固定后仍然可以选中它继续移动 / 缩放 / 旋转 / 四角变形。

## 五、缩放与平移（编辑大贴图时很有用）
- **滚轮** = 以光标为中心放大 / 缩小。
- **中键拖拽** 或 **按住空格拖拽** = 平移画面。
- 画布底部按钮：**－ / 适应 / 1:1 / ＋**。

## 六、调整图层
- 选中图层：拖动改位置，拖角缩放，拖到外面的圆把手旋转。
- 左下「图层」面板：调透明度、上移 / 下移层序、隐藏、删除（也可按 `Delete`）。
- **四角变形**：选中图层 → 工具栏「四角变形」→ 拖四个角做透视，再点「完成变形」。

## 七、导出（覆盖原 DDS）
1. 选好导出格式（默认 BC7·sRGB，一般彩色贴图用这个；不确定就选「跟随原图格式」）。
2. 点 **导出 DDS（覆盖原文件）** → 合并所有图层并写回 mod 里那张原始 DDS。
3. 工作目录里保留着**未合并、可续编**的记录和一份原图副本，所以导出后还能再打开继续改、重新导出，不会把图层叠加两次。

---

## 记录是怎么存的
```
<工作目录>/<mod文件夹名>/<贴图名>.dds       ← 原图副本（导出永不覆盖它）
                        <贴图名>.zzzproj    ← 分层记录，可继续编辑
```
开启自动保存后，编辑过程会自动写入；也可随时点工具栏「保存记录」。

## 自己重新构建 exe（开发者）
改完代码后，在项目根目录运行：`bash build.sh`
会重新编译 + 打包，并把最新的单文件 exe 覆盖到根目录的 `ZZZ-DDS-Editor.exe`。
